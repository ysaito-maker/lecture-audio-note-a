# アプリケーション仕様書

## ユーザー導線（最短導線）
1. APIキーを画面から入力（localStorageへ保存し、次回以降の入力を省略）
2. 講義名（任意）の入力
3. 音声ファイル（mp3, wav, m4a等）を選択
4. 実行ボタンの押下
5. 文字起こし全文の表示 ➔ 3行要約と要点の表示（連動処理）
6. 各結果のコピーボタンによるクリップボードへの保存

## エラー時の表示
- ファイル未選択時：アラート表示
- APIキー未設定時：アラート表示
- API通信エラー時：画面のステータス領域にエラー内容と対処法（短い音声で試す等）を表示
