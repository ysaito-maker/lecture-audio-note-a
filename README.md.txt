# 講義音声ノートAI (Gemini API連携)

## 概要
大学の講義や自習の録音をアップロードすると、Gemini APIを使用して自動で「文字起こし」「3行要約」「箇条書きの要点」を作成するスマホ対応Webアプリです。

## セットアップと使い方
1. 本リポジトリの `index.html` をブラウザで開くか、GitHub PagesのURLにアクセスします。
2. ご自身のGemini APIキーを入力し、「保存」を押します（localStorageに安全に保存されます）。
3. 講義名（任意）を入力し、30秒〜2分程度の短い音声ファイル（mp3/m4a等）を選択します。
4. 「AIでノートを作成する」ボタンを押すと、文字起こしと要約が自動生成されます。
